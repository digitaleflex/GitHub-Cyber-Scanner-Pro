# Code of Conduct

## Notre engagement

Nous nous engageons a rendre notre communaute accessible et respectueuse pour tous.

## Nos standards

### Comportements acceptables

- Utiliser un langage inclusif et respectueux
- Accepter les critiques constructives
- Se concentrer sur ce qui est le mieux pour la communaute
- Montrer empathie envers les autres membres

### Comportements inacceptables

- Le harcilement sous quelque forme que ce soit
- Les commentaires insultants ou derogatoires
- Les attaques personnelles ou politiques
- La publication d'informations privees sans consentement
- Toute autre conduite jugee inappropriée dans un cadre professionnel

## Application

Les cas de comportement inacceptable peuvent etre signales a l'equipe de maintenance via GitHub Issues.

## Application des consequences

L'equipe de maintenance a le droit de supprimer, modifier ou refuser tout contenu, commit, issue, ou PR qui viole ce code de conducte.

## Attribution

Ce Code of Conduct est adapte du [Contributor Covenant](https://www.contributor-covenant.org/fr/version/2/0/code_of_conduct/).
